# vlcovid
Backend Challenge 🏅 2022 - Covid Daily Cases

Projeto de back-end para obter dados dos datasets do kaggle.com sobre casos de Covid-19 e retorná-los como endpoints de uma API.

## Linguagens e Frameworks:
- Javascript
- Node.js
- PostgreSQL
- Knex
- Express
- Heroku
